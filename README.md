# HanSeekBar

自定义的SeekBar

1、可以定义一级二级跟踪颜色

2、可以定义等分分割和不登分分割

3、可以定义selection及bubble的字体大小及颜色

4、可以定义bubble动画显示的时长

5、可以定义bubble显示的内容

6、可以定义selection的显示内容及显示的位置
